Introduction

This is a Spring CPSC 415 - Data Vis. final class project. We chose to work with the Titanic data because it gives us a wide range to experiment and practice what we've done in class. There's a lot of analysis graphs and study samples that can be depicted from this data.

Note: We are trying our best to document how to use the platform. If you think that something is missing from the documentation, please do not hesitate to tell us about it. If you have any questions or issues regarding this theme, email us at ysemanag@trincoll.edu or gret@trincoll.edu

Getting started

You can directly use the compiled and ready-to-use the version of the platform.

Within the download you'll find the following directories and files, logically grouping common assets and providing both compiled and minified variations:

project1 /
├── project_dashboard/
      ├── css/
      ├── fonts/
      ├── images/
      ├── js/
      ├── pages/
      ├── partials/
      ├── index.html
      ├── scss/
      ├── vendors/
      ├── gulpfile.js
      ├── package.json
      ├── documentation/
├── CHANGELOG.md

Note: The root folder denoted further in this documentation refers to the 'project1' folder inside the downloaded folder
